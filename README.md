# Novel Gear - 執筆支援・ステータス管理エンジン
※本システムはGoogle Antigravityで開発されています。

**Novel Gear** は、小説のプロット、登場人物、そしてゲームシステムのような複雑なステータス設定をシームレスに管理するために設計された執筆支援アプリです。

## 🌟 主な機能

### 1. 登場人物管理（Character Management）

* キャラクターの基本情報（年齢、性別、外見、性格など）を一元管理 。


* ユーザーが自由に項目を追加・削除できる「追加属性」システムを搭載 。


* 入力と同時にデータが保存される「自動保存（Auto-Save）」機能により、執筆のフローを妨げません 。



### 2. 設定資料ライブラリ（World Building Library） 

* **ジョブ管理**: 初期値と成長率を設定し、レベルに応じたステータスを自動算出 。


* **スキル・装備品**: 実数（Flat）や割合（Percent）による動的なステータス補正をタグ形式で定義 。


* **世界観用語集（Glossary）**: 物語の固有名詞や設定を構造化して保存 。



### 3. 高度な動的ステータスエンジン 

* **自動計算ロジック**: `(基礎値 + SP) * (1 + %補正計) + 実数補正計` に基づくリアルタイム計算 。


* **時系列管理**: タイムライン（章・イベント）ごとにステータスや「その時のメモ」を記録 。


* **ステータスメモ**: 数値の変化だけでなく、イベントごとのドラマチックな変化を記述可能 。



### 4. 公開・エクスポート機能

* 読者に公開したい情報だけを抽出するプレビュー機能 。登場人物紹介やステータスを表示するときに使え機能。
