# 🛠️ Novel Gear - 次世代 創作支援スイート

**Novel Gear** は、小説家が「設定の矛盾」や「事務的な作業」から解放され、物語の核心に集中するために開発された統合執筆補助アプリです。

> ※ 本システムは **Google Antigravity** を活用し、AI と人間の共創によって開発されています。

---

## ✨ 主な機能（Features）

### 📊 動的ステータス算出エンジン
* **ステータスシミュレーター**: ジョブ、レベル、スキル、装備を組み合わせた最終数値を即座に算出。
* **時系列管理（タイムライン）**: 物語の進行に合わせたステータスの変遷を記録・プレビュー。
* **安全なフォールバック**: 削除済みのステータスも「(削除済：〇〇)」と表示し、データの不整合を視覚化。

### 📝 掲示板ライブ・エディタ
* **自動連番・リマウント**: レスの挿入・削除に合わせて番号を自動で振り直し。
* **名前ライブラリ連携**: 登録したキャラクター名やコテハンを引用し、一貫性のあるエピソードを作成。

### 🏗️ 設定資料ライブラリ (World Building Library)
* **統一された操作感**: ジョブ、スキル、装備、用語集をすべて共通の洗練された UI で管理。
* **動的プロフィール**: キャラクターごとに自由な設定項目を KVS 形式で追加・カスタマイズ可能。
