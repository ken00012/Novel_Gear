# Novel Gear - 執筆補助アプリ
※本システムはGoogle Antigravityを活用して開発されています。

**Novel Gear** は、小説のプロット、登場人物、そしてゲームシステムのような複雑なステータス設定をシームレスに管理するために設計された執筆補助アプリです。

🎯 こんな方にオススメ

1. **最強のステータス画面を作りたい！**
    
    複雑な計算はアプリに任せて、かっこいい設定作りに集中したいバトル小説家。

2. **掲示板回の編集で発狂したくない！**
    
    レス番号のズレやコテハンの管理に疲れ果てた、ダンジョン配信・掲示板のエピソードを書く作家。

###  1. Novel Gear：こんな時に大活躍します！

#### a) 「このキャラ、今レベル何だっけ？」と迷った時

* 物語の進行に合わせて、キャラクターの強さや装備をエピソード（第1話、第10話など）ごとに記録できます 。

* 「昔の装備に戻したい」「修行後のステータスを反映したい」といった時系列の管理が、クリック一つで切り替え可能です 。【改修中】


#### b) 「最強装備の補正、計算が面倒…」という時

* 手に入れた武器や習得したスキルをキャラクターに持たせるだけで、HPや攻撃力などの最終数値をアプリが自動で計算します 。

* 「15%アップ」などの複雑な割合計算もすべてお任せ。あなたは物語の展開を考えることに集中できます 。


#### c) 「物語の後半で設定が矛盾しそう」と不安な時

* 「起承転結」のボードで物語の全体像を俯瞰しながら、各シーンでのキャラクターの心の変化（アーク）をメモできます 。

* プロットの画面から、そのシーンのステータス画面へすぐ飛べるので、「この時点での登場人物のステータス」を瞬時に確認できます 。


#### d) 「掲示板回のレス番号を打ち直すのが苦行」な時

* 実際のネット掲示板に書き込むような感覚で、読者の反応エピソードを作成できます 。

* 途中に新しいレスを割り込ませても、番号（1, 2, 3...）は自動でズレなく振り直されます 。

### 2. 主な機能（Features）

* **動的ステータス算出エンジン**: ジョブ、レベル、スキル、装備を組み合わせ、最終ステータスをリアルタイムに自動計算します 。登場人物紹介のときにステータスを表示するときにコピペできるので便利です。

* **時系列データ統合システム**: 物語の初期設定を継承しつつ、特定のタイムライン（第1話、最終決戦等）ごとに独立したステータスや装備の上書きが可能です 。

* **起承転結プロットボード**: 物語の構造とキャラクターの感情変化（アーク）を可視化。各プロットから該当イベントのステータス画面へ1クリックでジャンプできます 。

* **掲示板ライブ・エディタ**: 実際の掲示板に書き込むような体験でレスを作成。自動連番、コテハン辞書、出力プレビュー機能を搭載し、投稿サイト用テキストフォーマットを自由に設定できます 。

* **設定資料ライブラリ（World Building Library）**：物語に登場するジョブやスキル、装備品、用語集を管理できます。
